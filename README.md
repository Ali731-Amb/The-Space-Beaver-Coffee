Welcome to **The Space Beaver Coffee**, an optional front-end challenge project where creativity meets web fundamentals — and where beavers apparently decided to conquer the galaxy with caffeine.

This website is a playful fictional concept built to practice the basics of modern front-end development. The goal is simple: create a clean, responsive, multi-page website using only core web technologies, while having fun with the design and storytelling.

No back-end.  
No database.  
No frameworks.  
Just pure HTML, CSS, and Vanilla JavaScript.

---

## 🌌 About the Project

The idea behind this site is to imagine a cosmic coffee shop inspired by space aesthetics and natural wooden vibes. The theme blends:

- A futuristic / intergalactic atmosphere  
- Warm, cozy, wood-inspired design elements  
- A lighthearted and creative narrative  

This project is intentionally fictional and flexible. The content (such as team members or product details) can evolve later — for now, the focus is on structure, layout, responsiveness, and clean implementation.

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla only)**

Strictly following the challenge constraints:
- No frameworks
- No backend logic
- No authentication system
- No database
- All content is hard-coded

The purpose is to strengthen foundational skills and understanding of how the web works at its core.

---

## 📄 Website Structure

The project is built as a multi-page website with shared layout elements.

### Header & Footer
- Present on all pages
- Consistent navigation structure
- Designed to maintain a unified visual identity

### Navigation Bar
- Links between all pages
- Fully responsive
- Includes a mobile-friendly transformation (such as a burger menu)

### Home Page
Includes:
- A hero section with a strong visual impact
- A short introduction to the concept
- A clear call-to-action

The homepage sets the tone of the project and introduces the overall atmosphere.

### Menu / Products Page
This section is designed to showcase items in a structured layout.

At this stage, the products are placeholders or conceptual — the focus is on:
- Using Flexbox or CSS Grid
- Creating a clean and organized visual structure
- Ensuring responsiveness

### Contact Page
Includes:
- A contact form (visual only)
- Required input fields
- JavaScript-based validation

The form does not send real emails. The objective is to practice structure and front-end logic.

### Team / Crew Section
A dedicated area to present the concept’s team.  
Since the crew details are not finalized yet, this section can remain flexible and evolve over time.
